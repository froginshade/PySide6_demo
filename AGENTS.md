# AGENTS.md

## Project Context

This repository is a personal learning workspace for PySide6.

Use it to create, experiment with, and organize small practice projects while learning PySide6 and desktop GUI development.

## Working Guidelines

- Prefer simple, readable examples that are suitable for learning.
- Keep each practice project self-contained when possible.
- Avoid introducing heavy frameworks or complex architecture unless the specific exercise needs them.
- When adding code, favor clear PySide6 patterns and short comments that explain non-obvious Qt behavior.
- Preserve existing exercises and user-created files unless explicitly asked to reorganize or remove them.
